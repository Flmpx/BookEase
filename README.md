### 一个本地化模拟购买和发布校园旧书的软件

#### 功能如下

##### 1. 登录与注册页

- 如果用户不存在, 可以通过输入学号和设置密码来创建一个新账号
- 如果用户存在, 通过输入学号和登录密码来进行登录, 若密码错误, 则无法登录


##### 2. 书籍购买页

1. 通过强大的检索能力来进行搜索书籍
	1. 检索条件包括但不限于卖家姓名, 学号, 书籍名称, 书籍ISBN等
	2. 点击开始检索即可查看筛选后的书籍
2. 点击一本自己有购买想法的书籍, 即进入书籍详情页
	1. 可以点击预定按钮来预定书籍, 预定后可与卖家联系
	2. 同时可以点击导出书籍信息到文件, 方便直接复制内容(Easyx图形库的局限性)


##### 3. 书籍发布页

- 通过修改书籍信息后进行发布


##### 4. 我的页

1. 我的信息页
	1. 可以修改自己的相关信息, 包括但不限于用户名, 电话等
	2. 可以修改自己的密码, 但需要进行二次验证
2. 我的发布页
	1. 可以查看自己的已发布书籍的在售, 已被预定, 已被购买和已下架的书籍
	2. 支持查看详细信息

3. 我的购买页
	1. 可以查看自己的预定和已购
	2. 支持查看详细信息
	





##### 5. 关于页

- 支持查看统计信息
- 包括但不限于软件走过的时间, 书籍的购买量, 总量等


#### 特点

1. 整个界面采用淡蓝偏白的风格
2. 按钮具备点击和鼠标悬浮特效
3. 在所有查看具有大量书籍的界面都具有以下功能
	1. 对书籍进行价格和发布时间的排序(已下架页不具备发布时间的排序)
	2. 界面左下角有翻页功能
	3. 右侧的主要书籍展示界面可以任意选择书籍, 同时具备鼠标悬浮和点击特效
4. 所有的书籍详情页都具备把书籍信息导出到文件的功能, 以便查看更加详细的书籍信息(由于Easyx只能进行图形绘制, 并不能进行文字拷贝, 故增加此功能)
